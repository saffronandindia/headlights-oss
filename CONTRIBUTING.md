# Contributing to Headlights

Thanks for your interest. This project is in pre-launch so the contribution surface is small but real.

## Reporting issues

Use GitHub Issues. Include what you tried, what you expected, what happened instead, and your Headlights + Python versions.

## Pull requests

- Fork, branch from `main`, submit PR
- Keep PRs small and focused
- Tests required for new behaviour
- All contributions licensed under Apache 2.0

## Code of Conduct

See [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md).
